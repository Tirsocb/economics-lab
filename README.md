# Juego Economico, Estructura de Datos
Tirso Cordova y Alejandra Lemus

## Iniciar
### Dependencias
1. Node
2. Express
3. Socket.io

### Instalar
1. Clonar proyecto y entrar al directorio
```bash
git clone <git del proyecto>
cd ./<directorio>
```
2. Instalar paquetes NPM
```bash
npm i
```
3. Correro
```bash
npm start
```# economics-lab
